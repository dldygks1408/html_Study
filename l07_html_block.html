html의 화면을 구성하는 block 요소들

상용 가능한 전체 넓이(width)이와 컨텐츠 만큼의 높이(height)를 갖는 사각의 박스로 여러 요소를 구조적으로 구성할 수 있다.

스타일로 본 block의 box model

boxModel
width : 150px
height : 150px
padding : 20px
border : 10px
margin : 40px auto
블럭을 정렬하는 auto

box Model 정리

  • box contents :
  • padding :
  • border :
  • margin :
  • block default(왼쪽 정렬) : margin : 10px 10px 10px auto
  • block 오른쪽 정렬 : margin : 10px 10px 10px auto
  • block 중앙 정렬 : margin : auto

Num 요소 (tagName) 설명 (description) 기본 스타일(default css)
01 < div > 모든 요소의 부모인 컨테이너로 요소를 구조적으로 만들때 사용 display : block;
02 < header > 제목, 저작권, 로고 작성자, 작성일 등을 표시하고 article의 헤드라인으로 사용한다. display : block;
03 < main > 문서에 하나 이기를 기대하며 페이지의 컨텐츠를 모두 포함하는 영역입니다. display : block;
04 < footer > 저작권, 연락처, 소셜미디어, 약관, 관련된 문서 display : block;
05 < section > 주제를 갖는 문서의 한 구문으로 여러 article를 포함한다. display : block;
06 < article > 작은 단위의 내용을 포함하는 구문 display : block;
07 < aside > 주변 콘텐츠와 간접적으로 관련되는 내용으로 화면의 사이드로 배치하는 편이다. display : block;
08 < nav > 탐색 링크의 집합인 navigaition으로 대체로 메뉴 리스트를 구성한다. display : block;
09 < address > 이메일 주소, URL 실제 주소, 전화번호, 소셜 미디어 같은 정보를 정의함 display : block; font-style : itailc
10 < form > 유저가 입력할 폼을 생성하는데 사용된다. display : block;
11 < figure > 그림, 도표, 사진, 코드 목록을 구성 display : block;
12 < blockquote > 다른 소스에서 인용 된 부분을 지정합니다. display : block;
13 < ul, ol, dl > 목록 구성하며 스타일 각각 다르다. list-style-type:(disc,decimal) display : block;
14 < p > 문서의 한 문단을 정의한다. display : block;
15 < h1~h6 > 문서의 제목이며 한 문단을 정의한다. h1은 검색에 활용됨으로 꼭 문서에 1개를 생성한다. display : block;
