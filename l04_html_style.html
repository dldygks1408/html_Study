HTML Style CSS (Cascading Style Sheets)

style css는 레이아웃, 색상, 텍스트, 크기, 이미지, 배경 RWD 등
HTML의 요소가 화면에 표시되는 모든 방식을 제공한다.

css 구문(Syntax)

  • selector{ property: "value"; (x) }
  • selector{ property: value; (o) }
  • 선택자{ 요소:값; 요소:값; 요소:값; ... }
  • 예) h1 { color : red; background-color : black; ...}
  • h1 { color : red;
    　　 background-color : black;
    　 }
  • tagname style="property = value; property = value;"

style HTML에 삽입하는 3가지 방법

  1. 외부 css : (head) (link rel="stylesheet" href="url") (/head)
  2. 내부 css : (head) (style) selector { property : value; } (/style) (/head)
  3. 인라인 css : (tagname style="property:value;") (/tagname)
  4. 동일한 스타일이 다르게 적용되었을때
