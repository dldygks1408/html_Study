HTML 규칙

  • 태그를 HTML의 요소라하며 태그는 시작과 끝이 있고
    <> 열고 < /> 닫는 다고 한다.
  • br, img, meta, input 태그는 내용이 없어서 닫지 않는다.
    br 닫는 것의 강박처럼 br/ 닫아주기도 한다.
  • 태그는 약속된 것만 사용하고 소문자로 쓰길 권장한다.
  • 이미지
  • img, a, meta와 같이 독특한 속성을 가지는 태그가 존재한다.
  • 루트 요소(html)를 가지는 구조적 형태를 tree 구조라 한다.
    html의 구조를 (Document Object Model) DOMtree 라 한다.
