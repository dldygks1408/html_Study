html SVG(Scalable Vector Graphics)

xml 형식의 벡터 기반 그래픽을 정의하는 요소

  • svg는 확장 가능한 웹용 벡터 그래픽을 나타낸다.
  • svg는 xml형식으로 그래픽을 정의하며 xhtml 형식을 쓴다. 예)열고 바로 닫는 요소가 "/>"
  • svg는 모든 요소와 모든 속성에 에니메이션을 정의 할 수 있다.
    (시간에 따라 스타일이 변하는 것을 에니메이션이라 한다.)
  • svg는 W3C html을 표준화하는 곳에서 권장하는 태그(요소)

SVG 그리기

    line, rect, circle, ellipse, polygon

  • width : 도형의 넓이
  • height : 도형의 높이
  • stroke : 테두리 색
  • stroke-width : 테두리 두께
  • fill : 도형에 색상 칠
  • opacity : 두명도
  • x: 사각형의 왼쪽 x좌표
  • y: 사각형의 왼쪽 y좌표
  • rx : 사각형 모서리 굴곡의 x축 반지름
  • ry : 사각형 모서리 굴곡의 y축 반지름
