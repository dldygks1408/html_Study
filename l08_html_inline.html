html의 inline 요소

inline 요소는 block에 포함된 요소로 넓이와 높이가 없으며 텍스트(contents) 영역에서 사용된다.

a, abbr, acronym, b, bdo, big, br, button, cite, code, dfn, em, i, img, input, kbd, label, map, object, q, samp, script, select, span, strong, sub, sup, textarea, tt, var

html 문자에 스타일을 지정하는 inline 요소

html 편집기의 띄어쓰기와 enter 인지해서 화면에 표시한다.
<b> 굵은 텍스트 font-weight : bold;
<strong> 굵은 텍스트(중요한 텍스트) font-weight : bold;
<i> 기울임 텍스트 font-style : italic;
<em> 기울임 텍스트(중요한 텍스트) font-style : italic;
<mark> 표시된 텍스트background-color yellow; color:black;
<small> 작은 텍스트font-size : smaller;
<del> 삭제된 텍스트text-decoration:line-through;
<ins> 삽입된 텍스트text-decoration: under-line;
<sup> 위 첨자 텍스트vartical-align:sup; font-size:smaller;
<sub> 아래 첨자 텍스트vartical-align:sub; font-size:smaller;

인용을 나타내는 inline 요소

<abbr> HTML약어 또는 두문자어를 정의
<bdo> abcdef 텍스트 방향을 정의
<cite> 작품의 제목을 정의
<q> 큰 따음표 인용 을 정의
