HTML Attributes

  • 모든 요소는 속성을 가진다.
  • 모든 요소는 id, class, title, style 속성을 가진다.
  • 모든 속성은 요소의 시작 태그에 작성한다.
  • 속성의 규칙은 name="value"로 작성한다.
    "value" or 'value' or value 로 표시 할 수 있다.
    웹표준은 "value"를 권장한다. (오작동을 막기 위해)
  • style은 html의 속성으로 css를 의미한다.
