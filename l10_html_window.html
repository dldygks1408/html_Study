Window (BOM) 객체는 브라우저에서 열린 창

script 에서 this로 취급되는 window는 최상위 객체다
window.document 로 document는 window 하위 객체고 이때 window는 생략 가능한다.
window는 브라우저의 모든 정보를 확인할 수 있는 Brower Object Model이다.

브라우저의 크기와 위치 정보(수정불가)

  • screenX, screenY(screenLeft, screenTop) : 브라우저의 x, y좌표
  • outerWidth, outerHeight : 모든 인터페이스 요소를 포함한 브라우저의 너비와 높이
  • innerWidth, innerHeight : 스크롤 막대를 포함한 창의 크기(뷰포트)

브라우저의 방문 기록(history)

window.history는 사용자가 방문한 URL을 포함한다.

  • history.lenth 히스토리 목록의 URL 수 (max : 50, 읽기전용)
  • history.back() : 히스토리 목록에서 이전 URL을 로드 (뒤로가기)
  • history.foward() : 히스토리 목록에서 다음 URL을 로드 (앞으로가기)
  • history.go(num) : 특정 기록목록에 URL로 로드

위치객체 location

window.location은 url정보가 포함되어 있다.

  • location.href : 현재 페이지를 반환하고 특정 페이지로 이동
  • location.hostname : 현재 url의 호스트 이름을 반환한다.
  • location.reload() : 현재 문서를 새로고침한다.
