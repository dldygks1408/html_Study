HTML class & id

모든 html 요소는 class 와 id 속성을 갖는다. 이때 두 속성은 선택자로 이용된다.

복수 선택 class

모든 html 요소에 하나 이상의 이름을 가지는 것을 클래스라 한다.
이때 클래스는 중복 가능하다.

  1. 규칙 : class="이름"
  2. 규칙 : class="이름1 이름2 이름3 ... " 이름을 여러개 선언 가능
  3. 규칙 : 선택자로 접근시 .이름{a : a;}
  4. javascript 접근 방법
    document.getElementsByClassName("className")[n]
    getElementS s가 붙어 배열형식
    class는 복수의 자료를(Document Object) 처리할 때 사용한다.

유일한 선택자 id

  1. 규칙 : id="idName" (o)
  2. 규칙 : id="id_name" (o x)
  3. 규칙 : id="name" (o x)
  4. 규칙 : id="01idName" (o x)
  5. 아이디를 만들때는 단어와 단어의 조합으로 시작은 소문자 조합되는 단어는 대문자로 작성한다.
    (o x) 작동은되지만 권장하지 않는 작성법
    권장하는 방식이 document에서 유일할 가능성이 높고 가독성이 높다.
  6. 규칙 : #idName{color : blue;}
  7. 규칙 : document.getElementById("idName")
